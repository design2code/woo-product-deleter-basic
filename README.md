# WooCommerce Product Deleter Basic
Yes yes, I know, you can just delete products from the Woocommerce admin panel.
I built this for a very specific use case where there were so many plugins, themes, and products that the Wordpress/Woocommerce admin barely functioned properly.
It is a simple lightweight WordPress plugin for managing and deleting WooCommerce products directly from your admin dashboard.

## Description

WooCommerce admin can be a bit slow on some hosts depending on shared hosting capabilities, plugins installed, themes used, buggy code and the amount of products. The Woocommerce Product Deleter Basic provides an intuitive admin interface to view, manage, and delete WooCommerce products. Whether you need to remove a single product or bulk delete multiple items, this plugin makes the process simple and efficient without loading any other data such as categories or tags assigned, SEO rankings, Pricing, Dates...etc.

## Features

- **Admin Dashboard Interface** - Dedicated management page under Tools menu
- **Single Product Deletion** - Delete individual products with AJAX
- **Bulk Product Deletion** - Delete multiple products at once
- **Real-time Product Count** - Check product count dynamically
- **Custom Admin Styling** - Polished user interface with custom CSS
- **AJAX-Powered Operations** - Fast, non-blocking product deletions

## Requirements

- WordPress 4.0 or higher
- WooCommerce 3.0 or higher
- PHP 5.6 or higher

## Installation

1. Download the plugin folder to your computer
2. Upload the `woo-product-deleter-basic` folder to `/wp-content/plugins/` directory
3. Activate the plugin through the WordPress Plugins menu
4. Ensure WooCommerce is installed and activated
5. Navigate to **Tools > Woo Product Delete** to access the plugin

## Usage

1. Go to the WordPress Admin Dashboard
2. Navigate to **Tools > Woo Product Delete**
3. View all available WooCommerce products
4. Select the product(s) you want to delete
5. Click the delete button to remove selected products

**Warning:** Deleted products cannot be recovered. Please ensure you have a backup before deleting products.
